# Paint Your Cloud in the Sky ☁️

一个创意的云朵艺术生成器，让你在蓝天画布上绘制和移动你的云朵！

## 功能特点 ✨

### 🎨 绘画功能
- **自由绘画**：在画板上自由绘制任何形状
- **可调画笔**：画笔大小可调节（5-50px）
- **撤回功能**：支持撤回最近的绘画操作（最多20步）

### ☁️ 云朵生成
- **智能转换**：将你的绘画自动转换为柔美的云朵
- **多层渲染**：3层云朵效果（外发光、主体、高光）
- **可拖拽移动**：每个生成的云朵都可以自由拖拽到任意位置

### � 天空背景
- **6种梦幻渐变**：精心设计的渐变天空背景
- **自定义上传**：支持上传你自己的天空照片作为背景

### 📐 输出尺寸
支持多种社交媒体和设备尺寸：
- Full HD (1920×1080)
- Instagram Square (1080×1080)
- Instagram Story (1080×1920)
- Facebook Post (1200×630)
- Twitter Header (1024×512)
- Desktop 16:10 (1920×1200)
- Desktop 2K (2560×1440)
- iPhone (750×1334)

### 🎮 操作按钮
- **GENERATE** - 将绘画转换为云朵
- **SAVE** - 保存作品为 PNG 图片
- **UNDO** - 撤回上一步绘画
- **DELETES** - 删除所有云朵
- **CLEARS** - 清空画板

## 使用方法 📖

1. **选择天空背景**
   - 点击样式按钮（1-6）选择预设渐变
   - 或点击相机图标上传自定义照片

2. **选择画布尺寸**
   - 从下拉菜单选择输出尺寸

3. **绘制图形**
   - 在右侧画板上绘制你想要的云朵形状
   - 调整画笔大小以获得不同效果
   - 可使用 UNDO 撤回

4. **生成云朵**
   - 点击 GENERATE 按钮生成云朵
   - 云朵会自动出现在天空画布上

5. **调整位置**
   - 在天空画布上拖拽云朵到理想位置
   - 可以生成多个云朵并自由排列

6. **保存作品**
   - 点击 SAVE 按钮保存为高清 PNG 图片

## 技术栈 🛠️

- **HTML5 Canvas** - 双画布系统（天空展示 + 绘画输入）
- **原生 JavaScript** - 无框架依赖
- **CSS3** - 渐变效果和响应式设计
- **拖拽 API** - 云朵移动功能

## 本地运行 🚀

1. 克隆仓库
```bash

cd print-your-cloud
```

2. 启动本地服务器
```bash
python3 -m http.server 8000
```

3. 在浏览器中打开

## 特色功能详解 💡

### 云朵渲染算法
采用多层渐变渲染技术：
1. 外发光层 - 营造柔和的光晕效果
2. 主体层 - 云朵的主要体积
3. 高光层 - 增加立体感和真实感

### 拖拽系统
- 自动计算云朵边界进行碰撞检测
- 支持多云朵重叠时的智能选择
- 流畅的鼠标和触摸事件响应

### 历史记录
- 自动保存每次绘画的状态
- 最多保存 20 步操作
- 支持逐步撤回
